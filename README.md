# Green Tech Hub — Landing Page

Landing Page giới thiệu dự án **Green Tech Hub** (kết nối giải pháp công nghệ bảo vệ môi trường, kêu gọi đối tác).

## Công nghệ

- HTML5 / CSS3
- Bootstrap 5 (CDN)

## Chạy project

Bạn có 2 cách:

### Cách 1: Mở trực tiếp

- Mở file `index.html` bằng trình duyệt.

### Cách 2: Chạy bằng server tĩnh (khuyến nghị)

Nếu bạn có Node.js:

```bash
cd D:\green-tech-hub
npx serve .
```

Sau đó mở URL mà terminal in ra.

## Cấu trúc thư mục

```
green-tech-hub/
├─ index.html
├─ assets/
```
